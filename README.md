# Portfolio Backend

Built with Express.js, MongoDB, Mongoose

## Tech Stack
- Express.js
- MongoDB + Mongoose
- Multer (file uploads)
- dotenv

## Getting Started
```bash
npm install
npm run dev
```

## Environment Variables
Create `.env` based on `.env.example`
